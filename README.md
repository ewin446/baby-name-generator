# SBTI 人格测试网站

一个有趣的MBTI风格人格测试网站复刻版。

## 功能特点

✨ **31道测试题** - 精心设计的人格测试题目
📊 **15维度分析** - 全方位的人格画像
🎭 **27种人格** - 包含各种有趣的人格类型
🔗 **一键分享** - 生成专属结果链接

## 快速开始

1. 直接在浏览器中打开 `index.html` 文件
2. 点击"开始测试"按钮
3. 回答31道问题
4. 查看你的人格类型和15维度分析
5. 分享你的结果给朋友

## 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式和动画
- **JavaScript** - 交互逻辑

## 本地运行

无需服务器，直接打开 `index.html` 即可运行。

## 自定义

### 修改题目
编辑 `script.js` 文件中的 `questions` 数组

### 修改人格类型
编辑 `script.js` 文件中的 `personalities` 数组

### 修改样式
编辑 `styles.css` 文件，自定义颜色、字体等

### 修改维度
编辑 `script.js` 文件中的 `dimensions` 数组

## 注意事项

- 当前版本的人格结果计算为简化版本（随机选择）
- 实际应用中应根据答案实现真实的计算算法
- 分享功能使用剪贴板API，需要浏览器支持
